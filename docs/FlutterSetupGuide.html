Setup Instructions

1
Install Flutter SDK from flutter.dev
2
flutter create welding_workshop_app
cd welding_workshop_app
3
dependencies:
  provider: ^6.1.1
  http: ^1.1.0
  file_picker: ^6.1.1
  fl_chart: ^0.65.0
  intl: ^0.18.1
4
flutter config --enable-web
5
flutter pub get
6
Copy code files into your project.
7
flutter run -d chrome

Firebase Deployment

1
npm install -g firebase-tools
2
firebase login
3
firebase init hosting
Select project: weldingworkshop-f4697
4
flutter build web --release
5
firebase deploy
